Background Image Properties

The CSS properties that are used to control background images are the background-repeat, background-size, background-position, background-origin, and background-clip. Each property has its own value within, which does something different for the image. The repeat section determines how an image lies within the title container, according to cssreferenceio. MDN Web Docs goes into the next three properties, including the background size which allows us to control the dimensions of the image, so we can keep it its original size or scale it to fit within borders. The next one is the position, which allows us to use options such as “top,” “bottom,” “left,” and “right” to move the positioning of the image on the website. The final is the origin, which specifies where the image starts so we can add padding and borders to the image itself. The background clip tends to get confused with origin and seem similar, according to a website called “teamtreehouse” which questioned if there was truly a difference between the two. The background-origin determines where the image starts, such as at the border or padding, while the clip tells us where the image ends; again, such as at the border or padding.
